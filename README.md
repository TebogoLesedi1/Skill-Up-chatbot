# Skill-Up-chatbot
A chatbot platform designed to help individuals showcase informal and self-taught skills, while sharing career insights to guide others into different fields.

## Problem

Many talented individuals have valuable skills but lack formal qualifications or platforms to showcase them. This makes it difficult for others to learn from their experiences or discover alternative career paths.

## Solution

Skill Up Chatbot provides a space where users can share their skills, experiences, and insights, helping others learn, grow, and explore different career opportunities.

## Tech Stack

- Python  
- Flask  
- HTML/CSS  
- JavaScript  
- LangChain  
- OpenAI API  
- tiktoken  
- Node.js (for localtunnel)
